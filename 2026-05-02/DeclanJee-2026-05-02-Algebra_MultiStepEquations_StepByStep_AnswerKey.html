Multi-Step Equations: Solve Line-by-Line

Student: Declan Jee Date: 2026-05-02
Teaching Goal: Show every step on its own line. Each line should change exactly one thing — distribute, combine, +/−, ×/÷.
Part A  ·  Two-Step Equations
Problem 1
\(4y + 7 = 23\)
Two-step Subtract first
\(4y + 7 = 23\) — given
\(4y = 23 - 7\) — subtract 7 from both sides
\(4y = 16\) — simplify
\(y = \dfrac{16}{4}\) — divide both sides by 4
\(y = 4\)
y = 4
Problem 2
\(-\dfrac{1}{3}b + 5 = 1\)
Two-step Fraction coefficient
\(-\dfrac{1}{3}b + 5 = 1\) — given
\(-\dfrac{1}{3}b = 1 - 5\) — subtract 5 both sides
\(-\dfrac{1}{3}b = -4\) — simplify
\(b = -4 \cdot (-3)\) — multiply both sides by −3 (reciprocal)
\(b = 12\)
b = 12
Problem 3
\(\dfrac{2n}{5} = -8\)
One-step Fraction coefficient
\(\dfrac{2n}{5} = -8\) — given
\(2n = -8 \cdot 5\) — multiply both sides by 5
\(2n = -40\) — simplify
\(n = \dfrac{-40}{2}\) — divide both sides by 2
\(n = -20\)
n = −20
Part B  ·  Combine Like Terms First
Problem 4
\(-3x + 8x = -25\)
Combine like terms Same side
\(-3x + 8x = -25\) — given
\(5x = -25\) — combine like terms (−3 + 8 = 5)
\(x = \dfrac{-25}{5}\) — divide both sides by 5
\(x = -5\)
x = −5
Problem 5
\(12 = 6m - 2 + m\)
Combine like terms Two-step
\(12 = 6m - 2 + m\) — given
\(12 = 7m - 2\) — combine like terms (6m + m = 7m)
\(12 + 2 = 7m\) — add 2 to both sides
\(14 = 7m\) — simplify
\(\dfrac{14}{7} = m\) — divide both sides by 7
\(m = 2\)
m = 2
Part C  ·  Distribute First, Then Solve
Problem 6
\(3(x - 2) = 18\)
Distributive property Two-step
\(3(x - 2) = 18\) — given
\(3x - 6 = 18\) — distribute the 3
\(3x = 18 + 6\) — add 6 to both sides
\(3x = 24\) — simplify
\(x = \dfrac{24}{3}\) — divide both sides by 3
\(x = 8\)
x = 8
Problem 7
\(-2(p + 5) = 14\)
Distribute negative Sign care
\(-2(p + 5) = 14\) — given
\(-2p - 10 = 14\) — distribute the −2 (watch the sign!)
\(-2p = 14 + 10\) — add 10 to both sides
\(-2p = 24\) — simplify
\(p = \dfrac{24}{-2}\) — divide both sides by −2
\(p = -12\)
p = −12
Part D  ·  Distribute + One More Step
Problem 8
\(4(k - 3) + 7 = -5\)
Distribute Combine constants
\(4(k - 3) + 7 = -5\) — given
\(4k - 12 + 7 = -5\) — distribute the 4
\(4k - 5 = -5\) — combine constants (−12 + 7 = −5)
\(4k = -5 + 5\) — add 5 to both sides
\(4k = 0\) — simplify
\(k = \dfrac{0}{4}\) — divide both sides by 4
\(k = 0\)
k = 0
Problem 9
\(5(c + 1) - 12 = -7\)
Distribute Combine constants
\(5(c + 1) - 12 = -7\) — given
\(5c + 5 - 12 = -7\) — distribute the 5
\(5c - 7 = -7\) — combine constants (5 − 12 = −7)
\(5c = -7 + 7\) — add 7 to both sides
\(5c = 0\) — simplify
\(c = \dfrac{0}{5}\) — divide both sides by 5
\(c = 0\)
c = 0
Problem 10
\(2d + 3(d - 4) = -27\)
Distribute Combine variables Hardest
\(2d + 3(d - 4) = -27\) — given
\(2d + 3d - 12 = -27\) — distribute the 3 (only inside parens)
\(5d - 12 = -27\) — combine like terms (2d + 3d = 5d)
\(5d = -27 + 12\) — add 12 to both sides
\(5d = -15\) — simplify
\(d = \dfrac{-15}{5}\) — divide both sides by 5
\(d = -3\)
d = −3
